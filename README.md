# Hostel Management System (Java + JDBC)

This is a console-based Hostel Room Management System.

## Features
- Add Room
- Allot Room
- Vacate Student
- View Room Status

## Tech Used
- Java (Core)
- MySQL
- JDBC

## How to Run
1. Import schema.sql into MySQL
2. Update DB credentials in DBConnection.java
3. Run Main.java
